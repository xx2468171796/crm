## 🚀 一键安装

### Windows 用户

```batch
双击运行: install-windows.bat
```

**安装选项:**
- ✅ Ask Continue MCP Server (必需)
- 🔧 Chrome DevTools MCP (可选，用于浏览器自动化)
- 🌐 本地/远程 Chrome 模式选择

### Linux / 远程SSH 用户

```bash
chmod +x install-linux.sh
./install-linux.sh
```

### 手动安装 VSIX

1. 打开 Windsurf
2. 按 `Ctrl+Shift+P`
3. 输入 `Install from VSIX`
4. 选择 `windsurf-ask-continue-1.1.0.vsix`
5. **重启 Windsurf**

## ✅ 安装完成后

### 基础使用

1. 开始任意对话
2. AI 完成任务后会自动弹窗询问"继续？"
3. 输入新指令，点击"继续"
4. 无限循环！

### Chrome 浏览器自动化 (如果安装)

**Windows 本地开发模式：**
- Chrome MCP 会自动管理浏览器
- 无需手动启动调试脚本
- 直接在 Windsurf 中使用浏览器工具即可

**远程开发模式 (SSH 到 Linux)：**
1. 在 Windows 运行 `start-chrome-debug.bat`
2. 记录显示的 IP 地址
3. 在远程服务器配置该 IP

## 🌐 远程浏览器调试 (SSH 开发)

如果你通过 SSH 连接远程服务器开发，需要在 Windows 本机启动浏览器调试：

### 步骤

1. **Windows**: 双击 `start-chrome-debug.bat`
   - 自动开放防火墙 9222 端口
   - 显示本机 IP 地址
   - 启动 Chrome 调试模式

2. **Linux 服务器**: 运行 `./install-linux.sh`
   - 选择 "1) 是 - 远程调试"
   - 输入 Windows IP 地址
   - 自动配置 chrome-devtools 和 playwright MCP

3. **重启 Windsurf**

### MCP 配置示例

**完整配置 (包含 Chrome MCP):**

```json
{
  "mcpServers": {
    "ask-continue": {
      "command": "python",
      "args": ["C:/path/to/server.py"]
    },
    "chrome-devtools": {
      "command": "npx",
      "args": ["chrome-devtools-mcp@latest"],
      "env": { "CHROME_CDP_URL": "ws://192.168.1.100:9222" }
    }
  }
}
```

**仅 Ask Continue (基础配置):**

```json
{
  "mcpServers": {
    "ask-continue": {
      "command": "python",
```

## ❓ 常见问题

### AI 不调用 ask_continue？

运行安装脚本后重启 Windsurf，规则会自动生效。

### 弹窗不出现？

检查 Windsurf 右下角状态栏是否显示 "Ask Continue: 运行中"。

### 远程 SSH 能用吗？

能！运行 `install-linux.sh` 即可。

### Chrome MCP 不工作？

1. **检查 Node.js:**
   ```batch
   node --version
   npx --version
   ```

2. **启动 Chrome 调试:**
   ```batch
   start-chrome-debug.bat
   ```

3. **检查连接:**
   ```batch
   curl http://127.0.0.1:9222/json
   ```

4. **重新配置:**
   ```batch
   cd mcp-server-python
   python install_mcp_config.py --with-chrome
   ```
