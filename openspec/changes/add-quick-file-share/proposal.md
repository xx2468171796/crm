# Change: 添加快速文件分享上传功能

## Why
客户需要一种便捷的方式向项目上传文件，无需登录门户即可通过分享链接快速上传。同时，桌面端技术人员也可以生成文件传输链接，方便与客户交换文件。

## What Changes
- **桌面端**：项目详情页新增"生成文件传输链接"功能，生成专属于该项目的上传链接
- **链接安全设置**：支持设置访问密码、访问次数限制、有效时间
- **分享节点**：集成后台分享节点配置，用户可选择不同区域节点生成链接
- **分享页面**：创建公开的文件上传页面，支持批量上传、文件夹拖拽上传
- **UI设计**：分享页面采用与客户门户一致的视觉风格（渐变背景、圆角卡片、现代配色）
- **存储逻辑**：上传的文件存储到对应项目的"客户文件"分类，自动重命名为"分享+文件名"
- **客户门户**：在项目详情中添加"资料上传"板块，客户上传的文件重命名为"客户上传+文件名"
- **桌面端同步**：上传的文件会出现在项目详情页的"客户文件"分类中

## Impact
- Affected specs: file-sharing (新增)
- Affected code:
  - `desktop/src/pages/ProjectDetailPage.tsx` - 添加生成链接按钮
  - `index/api/file_share_*.php` - 新建分享上传API
  - `index/public/share_upload.php` - 分享上传页面
  - `index/public/portal.php` - 添加资料上传板块
